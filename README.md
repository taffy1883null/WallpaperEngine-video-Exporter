#  Wallpaper Engine Exporter CLI

一个轻量、极简且高效的纯 C# 命令行工具，用于交互式扫描并导出 Steam Wallpaper Engine（壁纸引擎）中的视频壁纸

告别创意工坊中繁杂难记的“纯数字”文件夹，本工具会自动解析 `project.json`，提取壁纸的名称，并提供交互界面，能够勾选、批量导出 `.mp4` 视频源文件

---

##  快速上手
- 安装release里的exe即可

---
##  特性
- ** 纯交互式 CLI 界面**：支持上下翻页、单选、全选
- ** 智能名称解析**：自动读取 `project.json`，将文件夹ID输出为原作者命名的真实壁纸标题
- ** 安全导出与重命名**：
  - 自动过滤标题中不允许作为文件名的非法字符（如 `<>|?*`）
  - 智能处理重名文件，自动追加序号（如 `壁纸(1).mp4`），不覆盖原有文件
  - 100% 只读扫描，不修改、删除或污染你的 Steam 库资产
- **零外部依赖**：使用纯 C# 及原生的控制台 API 编写，无需额外安装第三方库

---

##  环境要求

- **.NET 8.0 SDK** 或更高版本
- 操作系统：Windows (涉及本地 Steam 默认路径操作)
